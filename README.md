# RADCARE NUSANTARA
Prototype frontend-only healthtech untuk radioterapi.

## Stack
React + Vite + TypeScript + Tailwind + React Router + Zustand + Recharts + Framer Motion + Lucide.

## Menjalankan
- npm install
- npm run dev
- npm run build

## Deploy Vercel
Framework: Vite, Build: npm run build, Output: dist.

## Role Demo
Pasien, Keluarga, Dokter Spesialis, Fisikawan Medis, Radioterapis, Perawat Navigator, Admin RS, Manajemen RS.

## Catatan
Semua data dummy lokal, simulasi localStorage, tanpa backend/database/API/auth asli.

## Roadmap Backend
Integrasi EMR/HIS, autentikasi SSO, audit trail real-time, message queue alert klinis, dan API interoperabilitas FHIR.
